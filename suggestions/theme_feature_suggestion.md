The theme could be extended to support additional YAML metadata fields — for example `orcid`, `bluesky`, and `github` — and render them automatically into a thank-you or contact slide, following the same pattern already used for `email`, `footer-author`, and `institute` on the title slide. This would mean any user of the theme only needs to populate their YAML front matter once to get a consistently formatted contact slide, without writing the layout by hand each time.
